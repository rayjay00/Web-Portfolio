Restaurant Site

I created this site for one of my favorite local restaurants. I used Bootstrap and compressed the images using Photoshop to help with page load speed. The most challenging part of this project was figuring out a way to have the dots between the menu items and prices scale up and down based on the size of the viewport. After some trial and error with some tricky CSS I finally figured it out. Hooray!

Reading Status Bar

Status bars help impatient people like me know how much more there is to the page. Instead of using a progress bar for an article, I chose to do it for a cool Edgar Allan Poe short story. I used jQuery to calculate the height of the window and the content. The toughest part about this project was re-calculating the status bar when resizing the viewport.

Tribute Page

I wanted to create a dynamic timeline page that popped in when scrolling down. I started off by sketching out what I wanted it to look like on phones, tablets, and desktop viewports. Next, I found the photos and used Photoshop to optimize them for the most efficient asset delivery. I used a customized jQuery framework to have it "pop in." My favorite part is the tiny George Washington head at the center of each section.
