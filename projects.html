Restaurant Site

The Idea: One of my favorite restaurants, Burrito Loco, needed a website. My idea was to create a single-page site that featured their menu, an about section, and a contact section complete with Facebook integration and practical information for customers.

The Solution: The most challenging part of this project was figuring out a way to have the dots between the menu items and prices scale up and down based on the size of the viewport. I figured out a way to solve this problem with some tricky CSS. Once that was accomplished, I focused on optimizing the site's images by creating a sprite sheet to improve page load speed.

What I Used: HTML5, CSS3, Bootstrap, Responsive Design

Ebook Progress Bar

The Idea: I have always liked news sites and eBooks that use progress bars to show your reading position on the page. They allow for the user to immediately judge the length of the article when scrolling. Instead of using a progress bar for an article, I chose to do it for a cool Edgar Allan Poe short story. In the future I would like to add some JavaScript to calculate the average reading pace of users and display about how long it will take to read.

The Solution: I hadn't used jQuery much before this project and I hadn't used the progress element at all. Eventually, I found some great resources to help me with this and calculated the height of the window and content using jQuery in order to allow the user to visualize their reading position.

What I Used: HTML5, CSS3, jQuery
